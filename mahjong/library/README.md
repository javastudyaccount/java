# mahjong4j
日本のルールにおける麻雀の役判定プログラムのJavaラッパーです

Androidで動いて欲しいのでJava7に対応させます

## NEWS!
全ての役をサポートしました！

# Contributing
手が出せるOSSを目指します

Pull Request・Issueはいつでもお待ちしております

## プルリクチャンス！
テストを書きましょう！

役判定を実際に試したテストを追加していただくのが一番簡単です

"/src/test/java/org/mahjong4j/SanankoTanyaoToitoihoTest.java"など[複数の役の名前]+'Test'<br>
をコピーして作りなおすのがオススメです。
